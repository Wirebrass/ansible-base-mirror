Ansible Role: workstation portage
=========

This role configure Gentoo Portage on a GNU/Linux workstation.

Requirements
------------

None.

Role Variables
--------------

All variables and default values are defined in `defaults/main.yml`.

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: localhost
      roles:
        - workstation_portage
