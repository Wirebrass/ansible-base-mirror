Ansible Role: workstation mutt
=========

This role configure Mutt on a GNU/Linux workstation.

Requirements
------------

Mutt, gpg2 and abook installed.

Role Variables
--------------

All variables and default values are defined in `defaults/main.yml`/

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: localhost
      roles:
        - workstation_mutt
